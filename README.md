<h3>The autonomous agent that rewrites your legacy enterprise software.</h3>

<i>This suite of specialized SKILL.md files is designed to ingest COBOL, Fortran, or legacy Java. It maps the underlying business logic via AST and autonomously rewrites the architecture into modern Go, Rust, or TypeScript without losing functionality.</i>
